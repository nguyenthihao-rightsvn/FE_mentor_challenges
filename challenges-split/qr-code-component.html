01

Box model

A really important thing to remember when writing CSS, is that everything displayed by CSS is a box. Whether that's a box that uses border-radius to look like a circle, or even just some text: the key thing to remember is that it's all boxes.

Content and sizing

The box content will affect the size of the box by default.
You can control this by using extrinsic sizing, or, you can continue to let the browser make decisions for you based on the content size, using intrinsic sizing.

The areas of the box model

Boxes are made up of distinct box model areas that all do a specific job.
The four main areas of the box model: content box, padding box, border box and margin box.

Controlling the box model

You also need to understand box-sizing, which tells our box how to calculate its box size.
By default, all elements have the following user agent style: box-sizing: content-box;.
You can use the alternative box model, border-box.

02

Selector

To apply CSS to an element you need to select it.
CSS provides you with a number of different ways to do this called selectors

The parts of a CSS rule

To understand how selectors work and their role in CSS, it's important to know the parts of a CSS rule.
A CSS rule is a block of code, containing one or more selectors and one or more declarations.

Simple selectors

  • Universal selector
  • Type selector
  • Class selector
  • ID selector
  • Grouping selectors

Complex selectors

  • Combinators
  • Compound selectors
