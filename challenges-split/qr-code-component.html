01

Box model

A really important thing to remember when writing CSS, is that everything displayed by CSS is a box. Whether that's a box that uses border-radius to look like a circle, or even just some text: the key thing to remember is that it's all boxes.

Content and sizing

The box content will affect the size of the box by default.
You can control this by using extrinsic sizing, or, you can continue to let the browser make decisions for you based on the content size, using intrinsic sizing.

The areas of the box model

Boxes are made up of distinct box model areas that all do a specific job.
The four main areas of the box model: content box, padding box, border box and margin box.

Controlling the box model

You also need to understand box-sizing, which tells our box how to calculate its box size.
By default, all elements have the following user agent style: box-sizing: content-box;.
You can use the alternative box model, border-box.

Selector

To apply CSS to an element you need to select it.
CSS provides you with a number of different ways to do this called selectors

The parts of a CSS rule

To understand how selectors work and their role in CSS, it's important to know the parts of a CSS rule.
A CSS rule is a block of code, containing one or more selectors and one or more declarations.

Simple selectors

  • Universal selector
  • Type selector
  • Class selector
  • ID selector
  • Grouping selectors

Complex selectors

  • Combinators
  • Compound selectors

03

The cascade

CSS stands for Cascading Stylesheets.
The cascade is the algorithm for solving conflicts where multiple CSS rules apply to an HTML element.
The cascade algorithm is split into 4 distinct stages.

  • Position and order of appearance
  • Specificity
  • Origin
  • Importance

Color

There are several different ways to specify color in CSS.
In this module we take a look at the most commonly used color values.

Hex colors

The hexadecimal ranges are 0-9 and A-F.
When used in a six digit sequence, they are translated to the RGB numerical ranges which are 0-255 which correspond to the red, green, and blue color channels respectively.
You can also define an alpha value with any numerical colors. An alpha value is a percentage of transparency.

RGB (Red, Green, Blue)

RGB colors are defined with the rgb() color function, using either numbers or percentages as parameters.

  • box shadow

05

Spacing

Find out the method of spacing elements.

Margin

If you want to add space to the outside of an element, you use the margin property.
The margin property is shorthand for margin-top, margin-right, margin-bottom and margin-left.
The margin shorthand can also be used with one, two, or three values. Adding a fourth value lets you set each individual side. These are applied as follows:

  • One value: all sides. (margin: 20px).
  • Two values: top-bottom and left-right. (margin: 20px 40px)
  • Three values: top and left-right and bottom. (margin: 20px 40px 30px).

Margin auto

A good example of auto margin is a horizontally centered wrapper which has a max width.
This sort of wrapper is often used to create a consistent center column on a website.

Padding

The padding property creates space on the inside of your box
